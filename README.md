<h2>❇️Flashing Method</h2>
<ul>
<li>Your phone's current Android version must be 16. If it is not, please first upgrade or downgrade to the official Google Android version 16.</li>
<li>If you are installing our ROM (LineageOS) for the first time, you must first flash three .img files.，</li>
<li>fastboot flash boot d:\YourDir\boot.img<br>
fastboot flash dtbo d:\YourDir\dtbo.img<br>
fastboot flash vendor_boot d:\YourDir\vendor_boot.img</li>
<li>Three of the .img files can be obtained using the following method:：payload-dumper-go.exe -o imgs -partitions boot,dtbo,vendor_boot payload.bin</li>
<li><a href="https://github.com/ssut/payload-dumper-go" target="_blank">payload-dumper-go</a> Please download it yourself.，payload.bin从ROM Unzip，`imgs` is the output directory generated by `payload-dumper`.</li>
<li>然后 fastboot reboot bootloader ，enter recovery，At this point, you should see the colorful Recovery interface. It is crucial to enter the Bootloader *before* entering Recovery; otherwise, the colorful interface will not appear, and the subsequent sideload process will fail.</li>
<li>Apply update<br>
Apply from ADB<br>
adb sideload d:\YourDir\lineage-xx.zip</li>
  </ul>

✅Flashing Method
👉Your phone's current Android version must be 16. If it is not, please first upgrade or downgrade to the official Google Android 16.<br>
👉If you are installing our ROM (LineageOS) for the first time, you must first flash three .img files.<br>
👉fastboot flash boot d:\YourDir\boot.img<br>
fastboot flash dtbo d:\YourDir\dtbo.img<br>
fastboot flash vendor_boot d:\YourDir\vendor_boot.img<br>
👉Three of the .img files can be obtained using the following method:：payload-dumper-go.exe -o imgs payload.bin<br>
<a href="https://github.com/ssut/payload-dumper-go" target="_blank">payload-dumper-go</a>Please download the necessary files yourself; extract `payload.bin` from the ROM, and note that `imgs` is the output directory generated by `payload-dumper`.<br>
👉 Next, execute `fastboot reboot bootloader` and then enter Recovery mode. At this point, you should see the colorful Recovery interface. It is crucial to enter the Bootloader *before* entering Recovery; otherwise, the colorful interface will not appear, and the subsequent sideload process will fail.<br>
